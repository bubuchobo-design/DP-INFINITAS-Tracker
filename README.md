# DP INFINITAS Tracker v1.0 Complete

iPhone / GitHub Pages 用の DP INFINITAS 管理アプリです。

## 収録内容

- INFINITAS DP向け
- ☆10 / ☆11 / ☆12 対応
- 曲名検索
- 難易度フィルター
- 曲名A→Z / 五十音ソート
- 難易度順ソート
- ランプ管理
- お気に入り
- 優先度
- メモ
- 難易度別進捗
- 今日の未HARD候補
- バックアップ / 復元

## GitHub Pagesでの導入

1. ZIPを解凍
2. 中のファイルをすべてリポジトリ直下にアップロード
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. Save
7. 発行されたURLをSafariで開く
8. 共有 → ホーム画面に追加

## ファイル構成

フォルダは不要です。

- index.html
- app.js
- style.css
- manifest.webmanifest
- icon.svg
- README.md

## 注意

ランプ・お気に入り・メモは端末のブラウザ内に保存されます。
機種変更やブラウザ削除の前に「バックアップ」を使ってください。
