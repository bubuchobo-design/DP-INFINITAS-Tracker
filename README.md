# DP INFINITAS Tracker v1.0

iPhone / GitHub Pages用のDP INFINITAS管理アプリです。

## 入っているもの
- ☆10 / ☆11 / ☆12
- 曲名検索
- 曲名A→Z・五十音ソート
- 難易度順ソート
- ランプ管理
- お気に入り
- メモ
- 統計

## 更新方法
このZIPを解凍して、以下のファイルをGitHubのリポジトリ直下に上書きアップロードしてください。

- index.html
- app.js
- style.css
- manifest.webmanifest
- icon.svg
- README.md

※ dataフォルダやtracks.jsonは不要です。
